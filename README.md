# Royal Painting - Kitchen Cabinets Painting Edmonton

A professional static website for Royal Painting, specializing in kitchen cabinets painting and refinishing services in Edmonton, Alberta.

## Features

- **SEO Optimized**: Focused on the keyword "kitchen cabinets painting edmonton"
- **Responsive Design**: Mobile-friendly layout that works on all devices

## Customization

### Adding Real Images

Replace the placeholder sections in the gallery and hero areas with actual photos:

1. **Hero Image**: Replace `.hero-placeholder` with actual kitchen cabinet photos
2. **Gallery Images**: Replace `.gallery-placeholder` with before/after photos
3. **About Image**: Replace `.about-placeholder` with team or workspace photos

### Updating Content

- Modify service descriptions in the Services section
- Update company statistics in the About section
- Add testimonials or reviews
- Update contact information if needed

## Local Development

To run locally:

1. Open `index.html` in a web browser
2. Or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   ```
